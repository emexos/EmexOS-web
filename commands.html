All about the shell

help

Shows a list of available shell commands in emexOS along with a short description.

        clear     - Clear the screen              |        info      - Show file info            
        about     - More about emexOS             |        rename    - Rename a file             
        echo      - Show messages on terminal     |        delfile   - Delete a file             
        shut      - Shutdown the system           |        ls        - Lists all files/dir       
        reboot    - Reboot the system             |        crdir     - Create a directory        
        oscheck   - Show system specs             |        deldir    - Deletes a directory       
        gfx       - Starts emexOS GUI             |        cd        - Changescurrent directory  
        crfile    - Create a file                 |        rfile     - Read file content

clear

This command clears the Screen

(no output)

echo Hello, World!

Displays the provided text in a new line in the terminal

example [SHELL]: 

 > echo Hello 
Hello 
 >

simply type in echo and then the text you want to show

shut

This command shuts down the system without any output.

(no output)

reboot

Reboots the system immediately. No output is displayed.

(no output)

gfx

Starts the emexOS graphical interface. If the mouse driver wasn't initialized, it gets activated automatically. A login window appears, and after setting a password, you are taken to the desktop environment.

[GUI is started, login screen shown]

crfile [filename]

Creates a new file with the given name in the current directory. No output is shown.

(no output)

crdir [dirname]

Creates a new directory in the current path. No output is shown.

(no output)

rfile [filename]

Displays the content of the specified file in the terminal.

[contents of the file]

info [filename]

Shows basic metadata of a file including name, size, and data offset.

            - info ---------------
            Filename:    [filename]
            Size:        [size in bytes]
            Data Offset: [offset value]
            ----------------------

rename [old] [new]

Renames a file. No output is shown on success.

(no output)

delfile [filename]

Deletes the specified file. No confirmation or message is displayed.

(no output)

deldir [dirname]

Deletes the specified directory. No output is shown.

(no output)

ls

Lists all files and directories in the current path.

[list of files and folders]

cd [directory]

Changes the current directory to the specified one. No output is shown.

(no output)

run [program]

Executes an ELF32 binary (with .sj extension) from the system if it exists.

[shows text/drawings from program]

init [component]

Initializes a system component (like Mouse or gfx). init gfx has the same effect as running gfx.

[Mouse initialized] or [GUI started]
