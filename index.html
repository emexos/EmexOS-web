What do you need, for a Mod-OS?

    if you want to make your own Mod-os simply download the OS and then install 

    " make, i686-elf-gcc, i686-elf-ld, nasm, and qemu "
    next step: run the Makefile in the terminal 'make',

    and if you don't want to install make then run ''./build.sh'.

    It doesnt work? bevore ''./build.sh' run 'chmod +x build.sh', but make sure that you are in the right directory

if you don't want to use qemu then you can use VirtualBox.
